Additive Manufactured Matching Medium Design for Implant Communications
Ozum Habiboglu, Sila Akgun, Erdem Cil, Ozlem Aydin Civi, Sema Dumanli
EuCAP, 2021
Presentation by Erdem Cil (youtube video) / Request for matlab GUI

Please click on header to read the abstract. Structure is designed and simulated in HFSS. Dielectric constant prediction algoritm is written in MATLAB. GUI is prepared using Appdesigner. Hexagons are printed using Makerbot 3D printer. Structure tested using VNA. Work done after paper is published: New reusable case is designed in Fusion360. Eigenmode simulations are updated in HFSS. All components are printed using Formlabs SLA printer. New structures are tested using VNA.

Design of soft surface to reduce coupling between patch antennas
Lukas Jonasson, Maria Pubill Font, Ozum Habiboglu
Applied Antenna Theory Course Final Project, 2022
Reference1 / Reference2

Theory and design in the references are studied. In CST, Soft surface EBG(Electron Bandgap) concept is applied to two element patch array operating in 6GHz. Antenna is manufactured by masking and etching. Farfield results are measured using near field scanner. Conclusion: EBG soft surfaces reduces coupling between patch antennas. Major lobe gain increased, side lobe levels decreased.
